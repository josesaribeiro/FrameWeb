Na pasta Gerador há o exe para gerar o modelo de navegação apartir
do arquivo local.frameweb ou passando parametro com o nome/caminho 
do arquivo para o .exe via DOS.

A pasta build é o resultado.

A pasta template é onde se encontra os modelos:

- framework -> tags JSF
- java -> Linguagem escolhida para as classes
- projeto -> templates base de cada projeto


Isso é ajusta no arquivo de configuração: GeradorFrameweb.exe.config
